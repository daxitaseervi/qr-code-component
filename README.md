## Overview
This is a solution to the Frontend Mentor QR code component challenge, a simple card built using HTML and CSS.

### Development process
I started by structuring the layout using HTML, placing elements in the order they should appear on the page and assigning classes to them. I used `div` containers to group related elements. After that, I styled everything using CSS step by step. I used Flexbox on the `body` to center the card using `align-items`. The project was initially built for desktop, but it turned out to work well on mobile screens without additional responsiveness.

### Challenges faced
Centering the card was initially confusing, and I overlooked using Flexbox on the `body`. After searching, I realized it was a much simpler solution than I expected. I also had trouble fixing the footer to the bottom of the page because I forgot about the `position: fixed` property. Once I revisited it, the issue was easy to solve. These issues made it clear that I need to revise core CSS concepts more consistently.

### Key learnings
- Better understanding of Flexbox for layout and centering
- Practical use of CSS positioning (`position: fixed`)
- Importance of revisiting fundamentals instead of overcomplicating solutions
